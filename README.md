📊 Global Superstore Sales Performance Dashboard
📌 Overview

This project showcases an interactive sales analytics dashboard built using Power BI on the Global Superstore dataset.
The dashboard provides insights into sales performance, profitability, order trends, and customer behavior across multiple years, categories, and regions.

🎯 Project Objectives

Analyze overall Sales, Profit, Orders, and Quantity

Track monthly sales and profit trends

Identify top-performing categories and sub-categories

Evaluate regional profit performance

Monitor order status distribution (Delivered, Pending, Returned)

Enable interactive filtering for business analysis
